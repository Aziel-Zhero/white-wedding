# **App Name**: Unidos para Sempre

## Core Features:

- Convite Digital: Página de convite de casamento com design clean e informações essenciais.
- Mapa Integrado: Exibição do local da cerimônia e recepção com mapa integrado do Google Maps.
- Menu de Navegação: Navegação intuitiva e fácil de usar para todas as seções do site.
- Lista de Presentes: Seção dedicada à lista de presentes de casamento, com links para lojas ou informações de contato.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke romance and tranquility.
- Background color: Off-white (#F8F8FF), a desaturated hue to contrast the primary color in a light scheme.
- Accent color: Pale pink (#F0B2B6), a subtle analogous hue to create gentle visual interest.
- Font pairing: 'Playfair' (serif) for headlines to set an elegant, fashionable tone, paired with 'PT Sans' (sans-serif) for body text, providing readability.
- Design limpo e responsivo, adaptado para diferentes dispositivos (desktop, tablets, smartphones) com TailwindCSS.
- Ícones temáticos relacionados a casamento e amor para guiar a navegação e destacar seções importantes.
- Transições suaves e animações sutis para melhorar a experiência do usuário ao navegar pelo site.
